Tools

Sat, Dec 20, 2025

Some tooling I created

Shift2Cal : Allows the user to create shift definitions (a name, start time and duration) which can then be applied to days after which all shifts can be exported to an ics file which can be imported in most calendar apps. This is something I created with Claude Code to help my wife, who as a nurse has a flexible schedule, working different days of the week and different shifts on those days. Lately the shifts have all been night shifts but the days on which she works in a given week vary.

hoeveelweken : A simple date calculator, created with a single Claude code prompt, just a quick tool to get the number of weeks to/from a date.
